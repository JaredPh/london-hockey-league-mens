# Hockey Club Website

A React-based website for displaying the london mens hockey league.

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured to automatically deploy to GitHub Pages when changes are pushed to the main branch.

The deployment workflow:
1. Builds the React application
2. Uploads the built files to GitHub Pages
3. The site will be available at: `https://[username].github.io/london-hockey-mens-league/`

## GitHub Pages Setup

To enable GitHub Pages deployment:
1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "GitHub Actions"
4. Push changes to the main branch to trigger deployment
